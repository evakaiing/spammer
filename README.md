# Spammer

Учебный проект из блока «Параллелизм» курса VK "Разработка веб-сервисов на Golang". Реализован потоковый конвейер, который имитирует обработку писем на спам. 


## Структура каталога

- `internal/service` - модели, статистика и mock-реализация внешних зависимостей.
- `internal/pipeline` - команды конвейера и тесты для модульной версии.


## Запуск тестов

```bash
go test ./...
```

## Покрытие

```bash
go test -coverprofile=cover.out ./...
go tool cover -html=cover.out -o cover.html
```
